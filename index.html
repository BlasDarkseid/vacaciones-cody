Mis vacaciones Frontend

Durante el verano de 2019, decidí comenzar a aprender desarrollo frontend, por lo que durante mi viaje, estuve aprendiendo HTML, CSS, y JavaScript

Cada tarde, después de conocer nuevos lugares en mis vacaciones, practicaba mis habilidades en tecnologías frontend, construyendo esta página

Estos son los cursos que estuve tomando:

Aprende cómo crear tu primera página web, no necesitas tener conocimientos previos de ningún tipo. Durante el curso te enseñaremos de manera práctica, y mucho más.

CSS es una poderosa tecnología que sirve para definir el diseño de una página web. Con CSS puedes crear animaciones, efectos, manejar colores, fuentes, y más.

Aprende JavaScript a profundidad, desde las bases del lenguaje, hasta el trabajo con objetos, programación asíncrona, novedades del lenguaje, etc.
